# Bitcoin Bottom Index Dashboard（中文版）

**在线页面：** https://brls123.github.io/BBI-Page/  
**English:** [View English README](./README.md)

这是一个面向公开访问的比特币市场仪表盘，用估值、情绪、宏观、链上和跨市场信号，来帮助判断当前市场所处的位置。

## 这个页面是干什么的

Bitcoin Bottom Index（BBI）想回答的是一个很实际的问题：

**当前市场更像是底部构建、BTC 修复、主升趋势，还是风险释放阶段？**

它不是靠单一指标下结论，而是把多个维度整合成一个更容易阅读的框架，让你看到的是“环境”和“结构”，而不是零散指标的堆砌。

## 页面上能看到什么

这个 dashboard 主要展示几层信息：

- **BBI 综合评分**：判断当前环境对底部形成和分批配置是否友好
- **市场状态分类（Regime）**：例如 Deleveraging、BTC Repair、BTC Bull、Alt Season
- **宏观 / 技术 / 基本面维度**：看强弱来自哪里
- **美股与山寨币广度背景**：看整体风险偏好和轮动状态
- **历史类比与情景比较**：把当前市场放回更长的历史坐标里理解

## BBI 是什么

**BBI = Bitcoin Bottom Index（比特币底部指数）**。

它是一个复合型分析框架，用来衡量当前环境对比特币“筑底”和“配置”的友好程度。

它**不是**用来精确预测某一根 K 线的最低点或最高点，也不是喊单工具。它更适合帮助你判断：

- 当前是不是处于历史上相对便宜的区间
- 市场是在继续释放风险，还是已经开始修复
- 当前环境是中性、偏强，还是已经过热

## BBI 由哪些部分构成

BBI 不是依赖一个指标，而是由多个类别的信号共同组成。

### 1. 估值（Valuation）
估值类指标用来判断，比特币相对历史水平是便宜还是昂贵。

页面上会涉及例如：

- **AHR999**
- **Mayer Multiple（梅耶倍数）**

### 2. 市场情绪（Sentiment）
情绪类指标帮助判断市场处于恐慌、极度恐慌、中性还是贪婪。

例如：

- **Fear & Greed Index（恐慌贪婪指数）**

### 3. 宏观环境（Macro）
比特币并不是独立交易的。流动性、利率、美元强弱、波动率和整体风险偏好，都会影响底部能不能真正站稳。

### 4. 周期 / 技术结构（Cycle / Technical Structure）
这一层主要帮助区分：当前到底是在破位、止跌、修复，还是重新回到更强的趋势结构中。

### 5. 链上背景（Onchain Context）
链上数据用来观察长期持有者行为、资金流向和网络状态，是偏支持、偏中性，还是偏弱。

### 6. 跨市场背景（Cross-market Context）
页面也会参考外部市场确认信号，例如：

- **美股 / 风险偏好**
- **山寨币广度**
- **BTC Dominance 与轮动状态**

因为有些阶段是 BTC 单独修复，有些阶段才会出现更广泛的扩散行情；也有些阶段整体仍然在去杠杆、降风险。

## 页面里的几种市场状态

为了更容易理解，dashboard 会把当前市场映射到几种更直观的 regime：

- **Deleveraging**：去杠杆、风险释放、广度偏弱
- **BTC Repair**：BTC 先修复，但整体扩散还不够强
- **BTC Bull**：趋势更顺、结构更健康、风险偏好更好
- **Alt Season**：BTC 之外开始出现更广泛的轮动和参与

## 这个页面适合用来看什么

如果你关心这些问题，这个页面会比较有用：

- 当前是不是更接近底部友好区间？
- BTC 是在修复，还是已经重新进入趋势？
- 山寨币到底有没有真正参与？
- 宏观和美股现在是在帮忙，还是在拖后腿？
- 当前环境更像过去哪些修复期，或哪些风险释放窗口？

## 说明

BBI 是一个分析框架，不构成投资建议。它的作用是帮助提升市场理解和决策质量，而不是保证结果。
